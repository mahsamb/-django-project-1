مستندات API جستجوی توصیفی

به مستندات API جستجوی توصیفی خوش آمدید.
با استفاده از API جستجوی توصیفی می‌توانید علاوه بر امکان جستجوی توصیفی، جستجو براساس مسخصات فردی، جستجو براساس نیازهای فردی و جستجو براساس تصاویر نیز داشته باشید.

📖 مستندات API جستجوی توصیفی

آدرس پایه (Base URL)


🔸آدرس پایه: 

http://YOUR_SERVER/api/search/


۱. جستجوی عادی (Normal Search)


🔸Endpoint:

POST /api/search/

🔸پارامترها:

ارسال به صورت application/x-www-form-urlencoded

پارامترنوع دادهاجباریتوضیحات
searchstring✔مقدار باید normal باشد
querystring✔متن جستجو
categorystring✔دسته‌بندی (تیشرت، شلوار، کیف، کفش)
SpeedAndAcuracystring✔تنظیم سرعت/دقت (S1A4 تا S4A1)
AI_Assistantstring✔نیاز به دستیار هوش مصنوعی (YesAI یا NoAI)

🔸پاسخ (Response):

{
  "query": "تی شرت مشکی",
  "results": [
    {
      "title": "تی شرت مردانه ساده",
      "description": "تی شرت نخی مشکی مناسب تابستان",
      "image_url": "123456",
      "category": "تیشرت"
    },
    {
      "title": "تی شرت یقه گرد",
      "description": "تی شرت طرح دار سفید",
      "image_url": "654321",
      "category": "تیشرت"
    }
  ]
}


۲. جستجو بر اساس مشخصات فردی (Character Search)


🔸Endpoint:

POST /api/search/

🔸پارامترها:

پارامترنوع دادهاجباریتوضیحات
searchstring✔مقدار باید character باشد
skin_colorstring✖رنگ پوست
eye_colorstring✖رنگ چشم
hair_colorstring✖رنگ مو
face_shapestring✖فرم صورت
body_shapestring✖فرم اندام
sizestring✖سایز کلی (لاغر، متوسط، چاق)
heightstring✖قد (بلند، متوسط، کوتاه)
SpeedAndAcuracy_Characterstring✔تنظیم سرعت/دقت
AI_Assistantstring✔نیاز به دستیار هوش مصنوعی

🔸پاسخ (Response):

{
  "full_description": "با توجه به ویژگی‌های شما، تی‌شرت نخی رنگ روشن مناسب‌تر است.",
  "results": [
    {
      "title": "تی شرت سفید ساده",
      "description": "مناسب برای پوست تیره و قد بلند",
      "image_url": "112233",
      "category": "تیشرت"
    }
  ]
}


۳. جستجو بر اساس نیازهای فردی (Requirement Search)


🔸Endpoint:

POST /api/search/

🔸پارامترها:

پارامترنوع دادهاجباریتوضیحات
searchstring✔مقدار باید requirement باشد
occasionstring✖مناسبت (تولد، مهمانی رسمی، مصاحبه و …)
weatherstring✖شرایط آب و هوا (گرم و آفتابی، بارانی، برفی و …)
SpeedAndAcuracy_PersonalRequirementstring✔تنظیم سرعت/دقت
AI_Assistantstring✔نیاز به دستیار هوش مصنوعی

🔸پاسخ (Response):

{
  "full_description": "برای مهمانی رسمی در آب و هوای بارانی، کت و شلوار تیره مناسب است.",
  "results": [
    {
      "title": "کت و شلوار مشکی",
      "description": "مناسب مهمانی رسمی و هوای بارانی",
      "image_url": "998877",
      "category": "شلوار"
    }
  ]
}

🔸 خطاها (Errors):

وضعیت (HTTP Code)توضیح
400 Bad Requestپارامتر اجباری ارسال نشده یا نامعتبر است
403 Forbiddenمشکل CSRF Token یا دسترسی
500 Internal Server Errorخطای داخلی سرور
